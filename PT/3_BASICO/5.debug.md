## Debug

_Debugs_ (também conhecido como Depuração) é uma forma de acompanhar, validar e também testar o seu código durante o desenvolvimento.

```csharp
if(playerLife > enemyLife){
    Debug.Log("Você pode matar um inimigo");
}
```
